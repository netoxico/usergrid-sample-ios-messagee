Messagee is a iPhone app using usergrid and reskit
==================================================

Requirements
------------

- OS X
- Xcode

Installation
------------

- git clone https://github.com/usergrid/usergrid-sample-ios-messagee.git
- cd usergrid-sample-ios-messagee
- git submodule update --init


Testing the App
---------------

- Open Messagee.xcodeproj
- Build and Run.
